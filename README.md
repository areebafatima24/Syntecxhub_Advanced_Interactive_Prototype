# 🏋️ FitPeak – UX Prototype

An interactive **fitness tracker prototype** designed to demonstrate smooth user flow, micro-interactions, and motivational feedback during workouts.

## 🎯 Overview

This project explores how **clear navigation and visual feedback** can improve user engagement in a workout app.

## 🔄 User Flow

Home → Workout Details → Processing → Workout Complete

Each step guides the user through a simple and motivating workout experience.

## ✨ Key Features

* Interactive multi-screen prototype
* Smooth transitions between screens
* Loading/processing feedback
* Workout completion confirmation

## 🛠 Tools

* Visily AI (UI Design)
* HTML / CSS / JavaScript
* GitHub
